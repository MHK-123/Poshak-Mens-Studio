# Poshaak – Luxury Tailoring Studio Website

A modern **static luxury website** created for **Poshaak – The Men's Studio**, a bespoke tailoring shop located in Secunderabad, Hyderabad.

This website showcases tailoring collections and provides customers with an easy way to **view the studio’s work and contact the shop for fittings**.

---

## Features

- Luxury **dark theme with gold accents**
- Modern **glassmorphism UI design**
- Clean layout built using **HTML, Tailwind CSS, and custom CSS**
- Collection showcase for premium suits
- Static **contact section with studio details**
- Fully **responsive design** for desktop and mobile

## Purpose of the Website

This website is **not an e-commerce store**.

It is a **static showcase website** designed to:
- Display tailoring collections
- Present the brand and studio
- Allow customers to contact the shop for fittings

All inquiries are handled directly through **phone, WhatsApp, or Instagram**.
